# Contributing to Steward Table

Thank you for your interest in Steward Table.

## Current Status: External PRs Paused

External pull requests are currently paused while we:
1. Complete LLC formation for the Steward entity
2. Establish CLA (Contributor License Agreement) infrastructure
3. Define contribution guidelines and code standards

**Copyright notice:** All code in this repository is currently copyright Emerson Ramos.
Upon LLC formation, intellectual property will be assigned to the Steward entity.

## For Now

- **Issues:** Please open issues to report bugs or request features. We read all of them.
- **Discussions:** GitHub Discussions is open for questions and ideas.
- **Security vulnerabilities:** Email security@steward.app (do not open a public issue).

## Planned Contribution Process

When we reopen external PRs, the process will include:
1. Sign the Steward CLA
2. Follow the coding standards in `docs/CONTRIBUTING_GUIDE.md` (coming soon)
3. Open an issue or discussion before large PRs
4. All PRs require passing CI (type-check, biome, vitest)

We look forward to building with the community.
